Earlsh Webtest
==============

A collection of headless browser tests written in JavaScript using [CasperJS](http://casperjs.org/) and [PhantomJS](http://phantomjs.org/) to automatically test my [Earlsh](https://github.com/fzuellich/earlsh) instance.

## How to run

    // install dependencies with npm
    npm install

    // run the bash script (don't forget chmod +x)
    sh run_tests.sh
